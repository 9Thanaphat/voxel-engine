# Voxel Game Project Rules

- walkthrogh หรือ plan ขอเป็นภาษาไทย
- ก่อนที่จะเขียนหรือแก้ไขโค้ดที่เรียกใช้ API ของ Bevy ให้ทำการค้นหาและอิงโครงสร้างจากโฟลเดอร์ `target/doc` ภายในเครื่องเสมอ เพื่อป้องกันปัญหา API เปลี่ยนแปลงในแต่ละเวอร์ชั่น
- ทุกครั้งที่แก้โค้ด ให้รัน `cargo check` เพื่อตรวจสอบความถูกต้องก่อนเสนอผลลัพธ์ให้ผู้ใช้เสมอ
